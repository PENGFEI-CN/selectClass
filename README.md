
# 选课
选课网站
=======
## 项目简介
    软件工程课设：预选课系统，自由选课的前提下，先获取全部的课程后根据学生选择本学期选择的课程名称和时间的筛选生成一个不发生在冲突的智能推荐的课程表。
## 项目采用主要语言
    前端使用vue以及bootstrap框架搭建网页，通过ajax请求后端的数据，后端通过python的Django框架搭建，数据库采用mysql
